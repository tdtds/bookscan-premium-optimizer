# BOOKSCAN Premium Optimizer

BOOKSCANのプレミアム会員向けのWebアプリケーション。
スキャンしたい本をリストアップすると、1箱に入れるべき組み合わせをページ数に応じて計算してくれます。

Herokuへpushするだけで動くようにデザインしてあります。
